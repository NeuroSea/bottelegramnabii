# 🤖 Trading Channel Bot

## Setup Instructions

### 1. Edit bot.py
Open `bot.py` and replace:
- `PUT_YOUR_TOKEN_HERE` → your token from @BotFather
- `your_username` → your Telegram username (without @)

### 2. Deploy on Railway (24/7 free hosting)

1. Create a free account at https://railway.app
2. Upload these 3 files to a GitHub repository:
   - bot.py
   - requirements.txt
   - Procfile
3. On Railway: New Project → Deploy from GitHub → select your repo
4. Done! Your bot runs 24/7 ✅

### 3. Your bot link
Once live, share this link everywhere:
```
https://t.me/YOUR_BOT_NAME
```
